# Tower Blocks Game 🧱🏗️

A thrilling 3D stacking game where precision and timing are key! Build the tallest tower by placing blocks perfectly on top of each other. Miss a placement, and watch your tower crumble! 🎮

## Features ✨
- Immersive 3D graphics powered by Three.js 🌐
- Smooth animations with GSAP 🎬
- Real-time score tracking 📊
- Responsive design for all devices 📱💻
- Simple controls: Click or press spacebar to place blocks 🚀

## How to Play 🎯
1. Click the "Start" button to begin.
2. Watch the block move back and forth.
3. Time your click or spacebar press to place the block on the previous one.
4. Stack as many as you can without missing!
5. If you miss, the game ends, and you can restart.

## Local Development 🛠️
To run locally:
1. Clone the repository.
2. Open `index.html` in a web browser, or use a local server (e.g., `python -m http.server 8000`).
